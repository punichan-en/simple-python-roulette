# 🎲 Simple Python Roulette
シンプルな Python ルーレットアプリケーション。
Tkinterを使用したGUIベースのルーレットゲームです。

🤖 LLM Journeyの一環です

## 🌟 特徴
- シンプルで使いやすいインターフェース
- スムーズなアニメーション効果
- 0から36までのランダムな数字生成

## 🛠 技術スタック
- Python 3.x
- Tkinter (GUI)

## 🚀 セットアップと実行方法

1. リポジトリのクローン:
```bash
git clone https://github.com/あなたのユーザー名/simple-python-roulette.git
cd simple-python-roulette
```

2. 依存関係のインストール:
```python
pip install -r requirements.txt
```
3. アプリケーションの実行:
```python
python src/main.py
```

# 🎮 使い方
1. アプリケーションを起動
2. "Start" ボタンをクリック
